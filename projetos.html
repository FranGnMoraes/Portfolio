Portal do Hyrax Desenvolvido

Projeto desenvolvido no âmbito da disciplina Design de Interfaces, com foco na construção de layouts responsivos utilizando CSS Flexbox.

Este site tem como temática os hyraxes — pequenos mamíferos sociáveis e carismáticos — e busca, através de uma estrutura visual clara e delicada, apresentar conteúdo de forma acessível, funcional e esteticamente harmoniosa. 🖋️📐

Objetivos

  • Aplicar os princípios do design responsivo com Flexbox;
  • Organizar conteúdos de forma coesa e hierárquica;
  • Criar uma navegação intuitiva, utilizando apenas HTML5 e CSS3;
  • Valorizar a estética com o uso de ícones, elements gráficos e composição visual.

Estrutura

O portal é composto por seções bem definidas, com títulos visuais enriquecidos por ícones e elementos gráficos suaves, criando um equilíbrio entre funcionalidade e identidade visual.

“Inspirado nos hyraxes: discretos, organizados e surpreendentes.” 🌿

Função: UX/UI Designer / Web Designer

Ferramentas: Figma, HTML, CSS (Flexbox), Bootstrap, JavaScript

EvenTech - Página de Eventos Em desenvolvimento

Projeto de uma plataforma de eventos desenvolvido no âmbito do programa Bolsa Futuro Digital. O objetivo é criar um portal intuitivo e responsivo para visualização, pesquisa e inscrição em palestras, workshops e eventos tecnológicos.

Atualmente na etapa inicial de estruturação, o projeto conta com semântica HTML5 e estilização básica em CSS3. O foco do desenvolvimento está na organização clara das informações, usabilidade e na construção de um layout limpo que facilite a experiência do usuário final. 📅💻

Objetivos

  • Criar um portal de eventos dinâmico, acessível e responsivo para múltiplos dispositivos;
  • Aplicar boas práticas de desenvolvimento web com HTML5 semântico e CSS3 modular;
  • Organizar a grade de eventos com filtros por categoria e datas para melhor usabilidade;
  • Estruturar a base do projeto para futura integração de lógica e interatividade com JavaScript.

Função: Desenvolvedora Front-end / UX/UI Designer

Ferramentas: HTML5, CSS3, Figma, Git/GitHub

Calculadora em React Desenvolvido

Projeto de uma calculadora funcional desenvolvido durante o curso de React da Rocketseat, com o objetivo de praticar e consolidar os principais conceitos da biblioteca. 🧮✨

Objetivo

Consolidar os fundamentos de React e praticar a construção de aplicações interativas com foco em organização de código e boas práticas.

Funcionalidades

  • Operações matemáticas básicas (soma, subtração, multiplicação e divisão);
  • Suporte a números decimais;
  • Limpar operação (C) e apagar último caractere (CE);
  • Exibição da operação e do resultado em tempo real;
  • Histórico de operações com persistência no LocalStorage.

Conceitos Aplicados

  • Componentes e Props;
  • Estado e eventos;
  • React Hooks (useState, useEffect, useContext);
  • Context API e Hooks customizados;
  • Renderização condicional e de listas.

Função: Desenvolvedora Front-end (React)

Tecnologias: React 18, JavaScript, Tailwind CSS, Git/GitHub

Como executar: Abra o arquivo index.html (ou utilize o servidor de desenvolvimento) em um navegador moderno.

Editor de Manchetes de Jornal Desenvolvido

Aplicação web interativa do tipo WYSIWYG ("What You See Is What You Get") desenvolvida para a disciplina de Design de Interação. O projeto fornece uma ferramenta visual que possibilita criar, diagramar e personalizar uma seção de notícias em tempo real com feedback instantâneo. 📰✍️

Objetivo

Fornecer uma ferramenta de edição e diagramação visual que permita customizar uma grade de notícias simulando um portal real de forma intuitiva, sem a necessidade de escrever código, aplicando princípios sólidos de usabilidade e acessibilidade.

Funcionalidades Principais

  • Interface em duas colunas: Painel do Editor (controles e formulários de estilo) à esquerda e Painel de Visualização (simulação real) à direita;
  • Personalização Geral: Controle da cor de fundo da seção e arredondamento de bordas gerais;
  • Imagem de Destaque: Upload por URL, configuração de texto alternativo (Alt Text) para acessibilidade e arredondamento personalizado;
  • Gerenciamento de Cards: Adicionar e remover cards dinamicamente, edição individual de títulos/subtítulos, estilo de fontes (negrito/itálico) e controle total sobre cores de textos, fundos e bordas.

Arquitetura & Engenharia de Software

  • Estado Centralizado (State): Um único objeto atua como a fonte única da verdade, armazenando dinamicamente todos os dados e configurações do editor;
  • Fluxo de Dados Unidirecional: Interações nos formulários atualizam o estado, que por sua vez dispara funções de re-renderização para redesenhar a interface;
  • Otimização com Delegação de Eventos: Os ouvintes de eventos do JavaScript são aplicados a containers pais para gerenciar cards dinâmicos com alta performance;
  • Google Fonts: Importação integrada das fontes Roboto e Playfair Display para simular a tipografia clássica de jornais impressos.

Função: Desenvolvedora Front-end / UX/UI Designer

Tecnologias: HTML5, CSS3, Tailwind CSS, JavaScript (Vanilla), Google Fonts, Git/GitHub

Como executar: Abra o arquivo index.html diretamente em seu navegador preferido.
