## 📌 Sobre o Projeto

Este repositório contém o código-fonte do meu **Portfólio Pessoal**. Desenvolvi este projeto para centralizar minhas principais criações de UX/UI Design, projetos de desenvolvimento front-end e registrar minha trajetória acadêmica e profissional. 

Como alguém que transita entre o design e a tecnologia, busquei refletir essa sinergia no design do site:
*   🎨 **Interface Moderna & Limpa**: Layout focado na legibilidade, usabilidade e contraste adequado.
*   🌓 **Theme Toggle**: Alternância suave entre modo claro (Light Mode) e modo escuro (Dark Mode).
*   📱 **Design Responsivo**: Adaptado para uma excelente experiência em smartphones, tablets e desktops.
*   📐 **Estrutura Semântica**: Construído com foco em boas práticas de SEO, acessibilidade e código limpo.

## 📂 Projetos em Destaque

Alguns dos principais projetos detalhados no meu portfólio:

1.  🌿 **Portal do Hyrax**  
    *Foco em design responsivo com CSS Flexbox e HTML5/CSS3.* Apresentação de conteúdo claro e esteticamente delicado sobre os pequenos mamíferos hyraxes.
2.  📅 **EvenTech (Página de Eventos)**  
    *Desenvolvido no programa Bolsa Futuro Digital.* Portal responsivo para visualização, pesquisa e inscrição em eventos tecnológicos.
3.  🧮 **Calculadora em React**  
    *Desenvolvido durante o curso da Rocketseat.* Aplicação funcional utilizando conceitos avançados de React (Hooks, Context API, LocalStorage).
4.  🌐 **Design de Interação (Consumo de APIs)**  
    *Estudo de interfaces dinâmicas.* Integração com serviços web, requisições assíncronas (`async/await`) e estados de carregamento amigáveis.
5.  📰 **Editor de Manchetes de Jornal**  
    *Aplicação WYSIWYG interativa.* Permite diagramar e personalizar seções de notícias em tempo real, usando estado centralizado com Vanilla JS e Tailwind CSS.
6.  🔗 **DevLinks**  
    *Agregador de links estilo Linktree.* Página responsiva baseada em Mobile-First com alternância de temas.

## 🚀 Como Visualizar Localmente

Se você deseja rodar este projeto em sua máquina local:

1. Clone o repositório:
   ```bash
   git clone https://github.com/FranGnMoraes/Portfolio.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd Portfolio
   ```
3. Abra o arquivo `index.html` em seu navegador ou use a extensão **Live Server** no VS Code para uma melhor experiência.
